\doxysubsection{Detailed Description}
Class hardware sim interface of the simulated Mia hand. 

This class inherits from public gazebo\+\_\+ros\+\_\+control\+::\+Robot\+H\+W\+Sim and it can be used also when the Mia hand is mounted on another robots that work with the gazebo\+\_\+ros\+\_\+control\+::\+Default\+Robot\+H\+W\+Sim class. In this case this class is able to control both the simulated mia hand and the other robot components.
